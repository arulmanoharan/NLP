from nltk import Tree

stanza.download("en")

def stanza_to_nltk_tree(stanza_tree):
    if isinstance(stanza_tree, str):
        return stanza_tree
    return Tree(stanza_tree.label, [stanza_to_nltk_tree(child) for child in stanza_tree.children])


def visualize_constituency_parse(doc):
    for sentence in doc.sentences:
        print("Constituency Parse Tree:")
        nltk_tree = stanza_to_nltk_tree(sentence.constituency)
        print(nltk_tree)
        nltk_tree.pretty_print()
        print("\n")
docs2 = [nlp2(sent) for sent in random_sent]
for doc in docs2:
    visualize_constituency_parse(doc)

On Comparing Dependency Parsing with Statistical(Constituency) Parsing, Dependency parsing is better in aspect for mapping dependency between text or sentences. whil Statistical was more focused towards phrase structure trees, Dependency parsing was focusing on grammatical relationships and their dependencies between the words in sentences or text. By this the time complexity was lower for dependency parsing compared to statistical. Also the Operations complexity was O(n^2) for dependency while compared to O(n^3) for statistical. Overall Dependency parsing was better than Statistical Parsing.
